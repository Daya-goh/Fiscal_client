## Fi$cal expense tracker

Technologies and Tools Used
- React
- Tailwind CSS
- Watch Mode API
- Insomnia

## Description
A personal day to day expense tracker to help users track their expenses and plan out their daily budget. Users can see the monthly overview of the different expenses and savings. There are many expense tracker in the market, however, most showed a monthly view. Hence, our team wanted to make an expense app for people who prefers to see the amount of money they can spend day to day.
